# Python

## Install

```bash
sudo pacman -S python
python --version
```

## Usage

```bash
nano test.py
print ("hello world")
```

## Tools

### Jupyter Notebook

```bash
pip install notebook --break-system-packages
jupyter notebook
```

You can also open `http://localhost:8888/tree` or `http://localhost:8888/lab` yourself
